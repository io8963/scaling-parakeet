## 3. 代码高亮测试 (重点)

### Python 代码 (带高亮)
应该显示彩色代码，背景为浅灰/深灰，且有圆角边框。

```python
def fibonacci(n):
    if n <= 0:
        return []
    elif n == 1:
        return [0]
    
    sequence = [0, 1]
    while len(sequence) < n:
        next_val = sequence[-1] + sequence[-2]
        sequence.append(next_val)
    return sequence

print(fibonacci(10))
# 这是一行注释
```

## 5. 提示块/警告块 (Admonitions)

由于你的配置开启了 `admonition` 扩展，以下块级元素应该有不同的颜色边框和背景。

!!! note "笔记 (Note)"
    这是一个普通的笔记块。通常用于补充说明信息。
    CSS 应该给它一个 **绿色** 的左边框。

!!! warning "警告 (Warning)"
    这是一个警告块。请注意检查配置是否正确。
    CSS 应该给它一个 **黄色** 的左边框。

!!! danger "危险 (Danger/Error)"
    这是一个错误或危险块。操作不可逆！
    CSS 应该给它一个 **红色** 的左边框。
